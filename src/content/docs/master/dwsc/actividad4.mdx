El objetivo de esta actividad es desarrollar un microservicio RESTful utilizando
Spring Boot. A diferencia de la actividad anterior donde se usó Jersey y WildFly,
en este caso se utiliza el framework Spring Boot con su servidor embebido Tomcat,
Thymeleaf para las vistas HTML y PostgreSQL como base de datos.

## 1. Inicialización del proyecto

El proyecto se crea utilizando [Spring Initializr](https://start.spring.io/),
seleccionando Maven como sistema de build, Java como lenguaje y la versión de
Spring Boot por defecto. Como metadatos del proyecto, se establece el grupo
`es.dwsc` y el artefacto `sampleprojectmicro`.

## 2. Ejercicio propuesto

El microservicio debe implementar las siguientes operaciones:

1. Una operación para obtener la lista de usuarios en formato JSON.
2. Una operación para visualizar la lista de usuarios en una tabla HTML.
3. Una operación para obtener los datos de un usuario particular en formato JSON.
4. Una operación para visualizar los datos de un usuario en una capa HTML.
5. Una operación para insertar usuarios.

## 3. Ejecución del proyecto

<Steps>

1. Iniciar la base de datos PostgreSQL

   Primero se inicia el servicio de PostgreSQL con el siguiente comando:

   ```bash
   docker-compose up -d
   ```

2. Ejecutar la aplicación Spring Boot

   A continuación se ejecuta la aplicación Spring Boot con el siguiente
   comando:

   ```bash
   ./mvnw spring-boot:run
   ```

3. Acceder a los endpoints

   Por último se pueden comprobar los endpoints:
   - JSON lista: [`http://localhost:8080/users`](http://localhost:8080/users)
   - Tabla HTML: [`http://localhost:8080/userstable`](http://localhost:8080/userstable)
   - Usuario JSON: [`http://localhost:8080/users/juan`](http://localhost:8080/users/juan)
   - Usuario HTML: [`http://localhost:8080/usertable/juan`](http://localhost:8080/usertable/juan)

</Steps>
