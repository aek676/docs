Bienvenido/a — este repositorio de notas y guías recoge las cosas que iré haciendo con tres objetivos claros:

- Reforzar el aprendizaje: consolidar conceptos y técnicas vistos.
- Tener una chuleta: disponer de resúmenes y ejemplos rápidos para consulta.
- Practicar la documentación de diseño: redactar decisiones, diagramas y explicaciones claras.

## Objetivos

- Registrar apuntes y resúmenes por asignatura y por tema.
- Incluir ejemplos de código y mini-proyectos prácticos.
- Documentar decisiones de diseño y justificaciones técnicas.
